# 🎓 Student Record Management System

A **Java Console-Based Application** to manage student records using **Core Java**, **Object-Oriented Programming (OOP)**, **Collections**, and **File Handling**. The application stores student information in a text file, ensuring data is retained even after the program is closed.

## 🚀 Features

* ➕ Add a new student
* 📋 View all student records
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete a student
* 💾 Automatically save records to a file
* 📂 Automatically load records when the application starts
* ⚠️ Exception handling for invalid inputs and file operations
* 🖥️ Menu-driven command-line interface

## 📌 Functionalities

### Add Student

Stores the following details:

* Student ID
* Name
* Age
* Course

### View Students

Displays all available student records.

### Search Student

Searches for a student using their unique Student ID.

### Update Student

Allows modification of an existing student's details.

### Delete Student

Removes a student record based on Student ID.

### Data Persistence

Student records are automatically saved to `students.txt` and reloaded when the application starts.

## ▶️ How to Run

### Using Eclipse

1. Open Eclipse.
2. Import the project.
3. Open `Main.java`.
4. Right-click → **Run As** → **Java Application**.

### Using IntelliJ IDEA

1. Open the project.
2. Open `Main.java`.
3. Click the **Run** button.

### Using Command Prompt

Compile the project:

```bash
javac *.java
```

Run the application:

```bash
java Main
```
